Fitur CMS Wordpress
CMS wordpress adalah cms yang mudah digunakan cocok untuk landing page dan website company profile yang sederhana.
  • Customisasi Landing Page / Halaman depan
  • Artikel / Post
  • Halaman / Page
Fitur CMS Wordpress + Woocomerce
Woocomerce adalah plugin yang digunakan pada CMS Wordpress yang memudahkan mengelola toko online.
  • Customisasi Landing Page / Halaman depan
  • Artikel / Post
  • Halaman / Page
  • Perhitungan Ongkos Kirim Barang
  • Laporan Penjualan
  • Notifikasi Email
  • Kontak / Hubungi Kami
  • Keranjang
Fitur CMS Basic Laravel

  • Halaman / Page
  • Gallery
